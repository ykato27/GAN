## GANとは

- GAN(Genera tive Adversarial Networks：敵対的生成ネットワーク)は、生成モデルの一種であり、データから特徴を学習することで、実在しないデータを生成したり、存在するデータの特徴に沿って変換できる手法です。
- GANは、イアン・グッドフェローらが2014年に発表した論文で、2つのネットワークを競わせながら学習させるアーキテクチャとして提案されました。
